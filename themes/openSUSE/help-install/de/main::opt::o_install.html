Installiereroptionen: install

Standardmäßig werden die lokalen CD-ROMs nach der Installationsquelle durchsucht.Bei einer Netzwerkinstallation wählen Sie die Option Installieren. MöglicheInstallationsprotokolle sind

  • FTP
  • NFS
  • HTTP

Die zu verwendende Syntax entspricht den Standard-URLs. Bsp.: Wenn Ihr Server bei192.168.0.1 gefunden wird und Sie eine eine NFS-basierte Installation aus demVerzeichnis /install durchführen wollen, bestimmen Sie die Quelle wie folgt:

  • install=nfs://192.168.0.1/install

Die Netzwerkkarte wird entweder mit dhcp konfiguriert oder Siemüssen die Parameter selbst bestimmen, wie beschrieben in Netzwerkoptionen.
