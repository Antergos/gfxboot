Test pamięci

Załączone Test pamięci zapewniają szerokie możliwości ustawień sprawdzania sprzętu. Głównym celem jest wykrycie uszkodzonej pamięci RAM, można również sprawdzać wiele innych elementów systemu.

Brak znalezionych błędów nie gwarantuje, że pamięć jest dobra, jednak większość błędów zostanie wykryta.
