Von Festplatte booten

Wählen Sie Installiertes Betriebssystem starten, um das auf Ihrer lokalen Festplatte installierteSystem zu starten. Dieses System muss richtig installiert sein, da nur derMBR (Master Boot Record) auf der ersten Festplatte gestartet wird. Die Geräte-IDder ersten Festplatte wird vom BIOS des Computers bereitgestellt.

Benutzen Sie dies, wenn Sie vergessen haben, die CD oder DVD aus dem Laufwerk zunehmen und den Computer von der Festplatte starten möchten.
