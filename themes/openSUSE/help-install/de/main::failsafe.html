Installation -- Sichere Einstellungen

Wählen Sie Installation -- Sichere Einstellungen, wenn der Computer während der Installation hängtoder nicht nachvollziehbare Fehler auftreten. Diese Option deaktiviert DMA fürIDE-Laufwerke und alle Energieverwaltungsfunktionen. Sehen Sie sich auch dieKernel-Optionen für apm, acpi und ide an.
