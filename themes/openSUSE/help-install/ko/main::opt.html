부트 옵션

부트 옵션으로 두가지 방식을 사용할 수 있습니다. 첫번째로, 설치 프로그램에 영향을 주는 옵션이고 두번째로, 커널 옵션 입니다. 일반적인 옵션은 다음과 같습니다:

a) 설치프로그램 옵션들

  • 설치 -- 설치 소스 선택
  • 네트워크 옵션 -- 네트워크 옵션
  • VNC 옵션 -- VNC를 통한 설치를 위한 옵션

b) kernel options

  • 스플래시 -- influence the behavior of the splash screen
  • apm -- toggle power management
  • acpi -- advanced configuration and power interface
  • ide -- control the IDE subsystem
