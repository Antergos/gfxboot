Speichertest

Der enthaltene Speichertest unterstützt gute Möglichkeiten die Systemhardwareauf Belastung zu testen. Der Hauptzweck ist die Erkennung von defektem RAM,es werden aber auch zahlreiche andere Systembereiche belastet.

Es gibt keine Garantie, dass der Speicher intakt ist, wenn keine Fehlergefunden werden, obwohl die meisten Speicherdefekte gefunden werden.
