Uso della Guida in linea

The boot loader online help is context sensitive. It gives information about the selected menu item or, if you are editing boot options, it tries to look up information about the option in which the cursor is positioned.

Navigation Keys

  • Freccia su: highlight previous link
  • Freccia giù: highlight next link
  • Freccia sinistra, Backspace: return to previous topic
  • Freccia destra, Invio, Spazio: follow link
  • Pagina Su: scroll up one page
  • Pagina Giù: scroll down one page
  • Home: go to page start
  • Fine : go to page end
  • Esc: leave help
